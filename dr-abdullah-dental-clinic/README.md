# Dr. Abdullah Dental Clinic Website

A premium, fully responsive, and bilingual (English/Arabic) dental clinic website with Dark/Light Mode support, built with Vanilla HTML, CSS, and JavaScript.

## Features
- **Bilingual Interface**: Seamlessly toggles between LTR (English) and RTL (Arabic) with Cairo, Outfit, and Inter Google Fonts.
- **Theme Toggling**: Dark and light mode options saved to local storage.
- **Before/After Split Image Slider**: Fully interactive comparative slider for cosmetic case studies.
- **WhatsApp Appointment Booking**: Form that compiles patient booking details and opens WhatsApp with a formatted message to `966555940208`.
- **Google Maps Integration**: Direct maps link pointing to the clinic location on Prince Abdulaziz Ibn Musaid Ibn Jalawi St, Riyadh.
- **Insurance and Finance Banner**: Lists approved insurance companies and installment options (Tamara, Tabby).

## Tech Stack
- HTML5
- CSS3 (Custom Variables, Flexbox, CSS Grid)
- JavaScript (Vanilla ES6, Intersection Observer API)
- FontAwesome Icons
